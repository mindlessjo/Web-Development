General HTML

Element : An HTML Element is anything from the start tag to the end tag

Links: Links are associated with the < a > tag.

HTML Styles :
  • Color, Background-color : Both change the color of the element
  • font-family, font-size : both change the appearance of the text
  • text-align : determines horizontal placement of text.

    • Format tags

    <>b : Enboldens the text

    <>i : Formats the text into italics

    <>mark : Highlights the text

    <>del : creates text with a line in it

    <>ins : underlines text withi the text

    General CSS

    Margins : Margins are used to create space around elements, outside of any defined borders.

    Outline : An outline is a line drawn outside the element's border.

    Border-style : The border-style property specifies what kind of border to display.

    Padding : Padding is used to create space around an element's content.

    Content : The content of the box, where text and images appear.

    CSS Height/Width

    auto : This is default. The browser calculates the height and width

    length : Defines the height/width in px, cm, etc.

    % : Defines the height/width in percent of the containing block.

    initial : Sets the height/width to its default value

    inherit : The height/width will be inherited from its parent value

    Alignment

    Justify-content : The justify-content property aligns the flexible container's items

    when the items do not use all available space horizontally.

    Align-item : Use the align-items property to align the items vertically.

    Console Commands

    * typeof : Used to determine the data type

    * let/const/var : used for setting and declaring values

    * alert() : Creates a notification in the top center of your screen

    * prompt() : Asks a question wit a provided response area that triggers an answer

    * console.log() : prints whatevers is stated into the web dev console

    Data types

    * Numbers : Used for decumals and integers/floating point numbers

    * String : Sequence of characters used for text

    * Boolean : Logical type that can only be true or false

    * Undefined : Sets a value as a variable but is not yet defined. ex:(let milk;)

    Number types

    * Null : means empty value

    * Symbol : value that is unique and cannot be changed

    * BigInt : Larger integers than the Number type can hold

    * let : used to declare variable and can be changed later

    * const : when defining a variable as a constant it can not be changed later
